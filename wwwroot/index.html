API for Electric Games

Endpoints

  • Get all: https://localhost:7012/game
  • Get from ID: https://localhost:7012/game/(id)
  • Get from Title: https://localhost:7012/game?title=(title)
  • Get Picture: https://localhost:7012/images/(picturename).jpg

How the data is in the API:

  • Id: int
  • Title: string
  • Platform: string
  • Releaseyear: string
  • Image: File | string
The different links above is to access the different games using different values.

When getting something from the API using the ID value from a game you would need to put in the URL: https://localhost:7012/game/3. This would make you get the game with the ID: 3.

When getting something from the API using the title value from a game you would need to put in the URL: https://localhost:7012/game?title=dragonBall. This would make you get the game with the the title dragonBall.

If you would want to get only the picture of a game you need to put in the URL: https://localhost:7012/images/dragonBall.jpg. This would mean that you need to know what the pictures name is saved in the wwwroot folder images. In this example we would get the dragonball picure
